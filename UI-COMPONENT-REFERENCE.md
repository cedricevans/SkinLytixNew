## ğŸ¨ Color & State Reference

### Colors
- **Blue:** Primary action (Next, Save buttons)
- **Green:** Success (success toast, counter 0-400 chars)
- **Amber/Yellow:** Warning (counter 400-500 chars)
- **Red:** Error/Danger (counter 500+ chars, disabled state)
- **Gray:** Disabled state, secondary text

### States

#### Button States:
- **Enabled:** Blue background, clickable, cursor pointer
- **Disabled:** Gray background, no interaction, cursor not-allowed

#### Input States:
- **Normal:** White background, gray border
- **Focus:** White background, blue border (thicker)
- **Error:** White background, red border, error message below

#### Radio/Select States:
- **Unselected:** Gray outline, empty
- **Selected:** Blue background/checkmark, highlighted
- **Hover:** Light gray background before click

## ğŸ§ª What Could Go Wrong (Troubleshooting)

### Issue: ReviewerAccuracyCard shows "No data available yet"
- **Cause:** No validations saved yet (first time user)
- **Expected:** After first validation, should show "1"
- **Action:** Continue testing, should update after save

### Issue: Word counter shows incorrect number
- **Cause:** Counting includes line breaks or special characters
- **Action:** Count manually, verify accuracy
- **Expected:** Should match basic word count (split by spaces)

### Issue: "Next" button stays disabled even after valid input
- **Cause:** Validation not detecting input
- **Action:** Verify text is in correct format
- **Expected:** Should enable immediately when valid

### Issue: Citation form doesn't add citation
- **Cause:** Validation failing silently
- **Action:** Check browser console for errors
- **Expected:** Should appear in list below

### Issue: Save produces error toast instead of success
- **Cause:** Database save failed
- **Action:** Check database connection
- **Expected:** Should save immediately

### Issue: Modal/panel won't close after save
- **Cause:** Form didn't reset properly
- **Action:** Refresh page manually
- **Expected:** Should close and return to product list
